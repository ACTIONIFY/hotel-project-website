Adding Images to Your Website

To add an image to your website, use the <img> element. The src attribute specifies the path to the image file, and the alt attribute provides alternative text for the image.

<img src="image.jpg" alt="A description of the image">

Example Image

You can also specify the size of the image using the width and height attributes. These attributes accept a value in pixels or as a percentage of the available space.

<img src="image.jpg" alt="A description of the image" width="200" height="100">
