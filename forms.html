Creating Forms in HTML

To create a form in HTML, use the <form> element. The action attribute specifies the URL of the server-side script that will process the form data, and the method attribute specifies the HTTP method used to submit the form data (e.g., "get" or "post").

<form action="/process-form.php" method="post">
  ...form elements go here...
</form>

Example Form



There are many different form elements that you can use to collect different types of data. Some common form elements include:

  • <input type="text">: a single-line text field
  • <textarea>: a multi-line text field
  • <select>: a dropdown menu
  • <input type="radio">: a radio button
  • <input type="checkbox">: a checkbox
