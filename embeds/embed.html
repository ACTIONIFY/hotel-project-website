Using Embeds in HTML

The <embed> element allows you to embed media (such as audio or video) in an HTML page. To create an embed, use the <embed> element and specify the URL of the media file using the src attribute.

<embed src="http://www.example.com/media.mp3">

Example Embed

You can also specify the size of the embed using the width and height attributes. These attributes accept a value in pixels or as a percentage of the available space.

<embed src="audio.mp3" width="400" height="200">
