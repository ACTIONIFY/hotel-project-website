Using Iframes in HTML

An iframe (inline frame) is an HTML element that allows you to embed another HTML document within the current page. To create an iframe, use the <iframe> element and specify the URL of the document to be embedded using the src attribute.

<iframe src="http://www.example.com"></iframe>

Example Iframe

You can also specify the size of the iframe using the width and height attributes. These attributes accept a value in pixels or as a percentage of the available space.

<iframe src="http://www.example.com" width="800" height="600"></iframe>
