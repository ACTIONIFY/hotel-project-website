Inserting Tables

Tables can be used to organize data in a grid format. To insert a table, use the <table> element.

The table rows are defined with the <tr> element, and the cells are defined with the <td> element. You can also use the <th> element to define table headings.

<table>
  <tr>
    <th>Column 1</th>
    <th>Column 2</th>
  </tr>
  <tr>
    <td>Row 1, Cell 1</td>
    <td>Row 1, Cell 2</td>
  </tr>
  <tr>
    <td>Row 2, Cell 1</td>
    <td>Row 2, Cell 2</td>
  </tr>
</table>
